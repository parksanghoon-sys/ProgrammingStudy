# 크롬 개발자 도구 명령어

* 요소 템열기
  * Ctrl + Shift + C
* 콘솔 탭 열기
  * Ctrl + Shift + J
* 실행
  * Ctrl + Shift + P
* 로컬 스토리지
  * 저장하기
    * localStorage.setItem("key","value")
  * 불러오기
    * localStorage.getItem("key")
  * 삭제하기
    * localStorage.removeItem("key")
  * 모두 삭제하기
    * localStorage.clear()
  * 키/값 쌍의 개수
    * localStorage.rlength())
