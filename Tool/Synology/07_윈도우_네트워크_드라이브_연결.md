# 윈도우에서 Synology NAS를 네트워크 드라이브로 연결하기

## 1. 사전 준비
- Synology NAS와 윈도우 PC가 같은 네트워크(또는 VPN)에서 접근 가능해야 합니다.
- Synology DSM에서 `제어판 > 파일 서비스`의 `SMB`가 활성화되어 있어야 합니다.
- 접속할 공유 폴더(예: `share`)와 사용자 계정 권한이 설정되어 있어야 합니다.

## 2. 네트워크 드라이브 연결 (파일 탐색기)
1. 윈도우 `파일 탐색기`를 엽니다.
2. 왼쪽에서 `내 PC`를 선택합니다.
3. 상단 메뉴에서 `...` 또는 `컴퓨터` 탭의 `네트워크 드라이브 연결`을 클릭합니다.
4. 드라이브 문자(예: `Z:`)를 선택합니다.
5. 폴더 경로에 아래 형식으로 입력합니다.

```text
\\NAS_IP\\공유폴더명
예) \\192.168.0.20\\share
```

6. 필요하면 `로그온할 때 다시 연결`을 체크합니다.
7. 다른 계정으로 접속 시 `다른 자격 증명을 사용하여 연결`을 체크합니다.
8. `마침` 클릭 후 Synology 계정(ID/PW)을 입력합니다.

## 3. 연결 확인
- 파일 탐색기 `내 PC`에 지정한 드라이브 문자로 NAS가 표시되면 완료입니다.
- 읽기/쓰기 권한이 올바른지 파일 하나를 생성/삭제해 확인합니다.

## 4. 자주 발생하는 문제 해결
- `네트워크 경로를 찾을 수 없습니다`:
  - NAS IP가 변경되었는지 확인
  - 방화벽/공유기에서 SMB(기본 445 포트) 차단 여부 확인
- `액세스 거부`:
  - DSM에서 해당 사용자의 공유 폴더 권한 확인
  - 윈도우에 저장된 이전 자격 증명 삭제 후 재시도
- 연결이 자주 끊김:
  - 고정 IP 설정 권장
  - 절전 모드/네트워크 어댑터 전원 관리 옵션 확인

## 5. 빠른 연결 명령 (선택)
명령 프롬프트 또는 PowerShell에서 아래처럼 연결할 수 있습니다.

```powershell
net use Z: \\192.168.0.20\\share /user:NAS아이디 비밀번호 /persistent:yes
```

연결 해제:

```powershell
net use Z: /delete
```
