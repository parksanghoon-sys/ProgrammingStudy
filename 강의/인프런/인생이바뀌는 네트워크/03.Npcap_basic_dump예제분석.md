## 주요 함수

* pcap_findalldevs()
  * H/W Nic 인터페이스를 고르는 부분
  * NIC 가 몇개가 있는지 어떤게 있는지 확인하는 부분
  * Device GUID 장치 이름을 가져온다.
* pcap_open_live()
  * Device 장치를 Open
  * Network 데이터 정보를 읽는부분
* **pcap_loop()**
  * L2 Frame 단위 데이터가 들어오고 나오는것을 감시하는 여할
  * Packet_handler() 함수 Callback
* pcap_freealldevs()
  * release pcap_findalldevs
* pcap_close()
